# Receino（レシーノ） ![Tier](https://img.shields.io/badge/tier-utility-orange)

> Scan. Know. Save. — 撮って、知って、節約する。

## これは何？

レシートを撮るだけで、AI が品目・カテゴリ・金額を自動で読み取り、食費を見える化する家計簿アプリです。手入力なしで「何にいくら使っているか」がわかり、節約の余地まで提案します。

## こんな時に使えます

- 食費をざっくりでも把握したい・減らしたいとき
- レシートの手入力が面倒で、家計簿が続かないとき
- 家族の食費を共有して、無駄な出費を見直したいとき

## 使い方

- 🌐 デモ: https://receino.com
- レシートを撮影 → AI が自動で分類 → 保存するだけ。月の食費・カテゴリ別の内訳・年間の見込みが自動で見えます。
- 初回は5枚まで無料。アカウント登録で毎週 +1 枚、友達紹介でお互いに +5 枚もらえます。
